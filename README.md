# AssignmentTracker

A simple application designed to help you manage assignments by breaking them down into subgoals, tracking progress, and managing time.

## Features

* **Subgoal Checklist:** Create a checklist of subgoals for your assignments.
* **Progress Tracking:** Monitor your progress with a visual progress bar.
* **Time Management:** Track time spent on tasks with a built-in timer and keep an eye on the current time with a real-time clock.
